## Обзор датасета

- **Источник данных:** [IMDB Movie Dataset](https://github.com/shishir349/Analyzing-the-IMDB-Movie-Dataset/blob/master/movie_metadata.csv)
- **Количество фильмов:** 5043
- **Период выпуска фильмов:** 1916–2016 гг.

### Структура данных
| Категория | Столбцы |
|-----------|---------|
| **Основная информация** | Название фильма, год выпуска, продолжительность, язык, страна производства, возрастной рейтинг |
| **Жанры** | 26 уникальных значений (Action, Adventure, Comedy, Drama, Romance, Sci-Fi, Thriller и др.) |
| **Рейтинг** | Рейтинг IMDB, количество оценок |
| **Финансовые показатели** | Бюджет, сборы |
| **Участники** | Режиссер, актеры (первые 3 в титрах) |
| **Социальные метрики** | Количество лайков на Facebook (для актеров, режиссера и фильма в целом), количество рецензий критиков и отзывов пользователей |
| **Технические характеристики** | Цвет изображения, соотношение сторон кадра|

## Рекомендательная система на основе кластеризации K-means

### Очистка и масштабирование данных
- Для наглядности удален первый квартиль по количеству оценок (1261 фильма с менее 8594 оценок)
- Далее проведена очистка от пропусков
- Признаки масштабируются методом `StandardScaler()`

## Выбор оптимального количества кластеров (k)

Для определения оптимального числа кластеров использовались метрики:

1. **Inertia** - сумма квадратов расстояний внутри кластеров (чем меньше, тем лучше)
2. **Silhouette Score** - мера компактности и разделимости кластеров (чем выше, тем лучше)

### Анализ результатов

<img width="1158" height="451" alt="image" src="https://github.com/user-attachments/assets/547489b8-ebaf-494d-89f7-e00b63394a8d" />


Основываясь на методе локтя (elbow method), оптимальное значение k может быть равно 13, 16 или 19. 
Однако по Silhoette score наибольшие значения достигаются при k=24, k=25, k=21. 
В матрице признаков 20 жанров, поэтому распределение на ~20 кластеров может быть оправдано, однако чем больше значение k, тем выше вероятность переобучения. 
Выберем **k=17** в качестве оптимального, так как при нем значениe Silhouette score больше, чем для соседних k, и превышает 0.2.

Анализ средних значений жанровых признаков по кластерам показывает, что кластеризация успешно выделяет фильмы со схожими жанровыми комбинациями.

- На тепловой карте можно заметить, что более 75% фильмов каждого кластера относятся к одному или 2-4 определенным жанрам. Это свидетельствует о высоком уровне интерпретируемости полученных кластеров.
